# home-cn-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何具体域名或网站，旨在提供通用的资源展示或信息承载功能。

## 项目简介

这是一个静态资源仓库，主要存放自包含的 HTML 文件。每个页面均为独立设计，不依赖外部数据库或后端服务，适合用于离线查看、本地部署或简单信息展示。

## 目录结构

仓库根目录下按类型或用途组织文件，目前包含：

- `pages/` — 主要的 HTML 页面存放目录
- `assets/` — 页面引用的样式、脚本、图片等资源文件
- `docs/` — 相关说明文档或附加资料（如有）

具体文件列表请参见仓库文件索引。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，内部引用的资源尽量使用相对路径。
- 页面内容以中文为主，部分可能包含少量英文术语。
- 页面不包含跟踪代码、广告或第三方远程资源（除非特别说明）。
- 归档的页面可能不定期更新或补充，不保证所有页面长期可用。
